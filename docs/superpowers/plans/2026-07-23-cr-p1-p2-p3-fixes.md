- [ ] **Step 4: 修改安装脚本**

在临时文件下载阶段增加：

```sh
binary_url="$gh_base/aris-$os-$arch"
checksum_url="$binary_url.sha256"
checksum_tmp=$(mktemp "${TMPDIR:-/tmp}/aris-checksum.XXXXXX")
trap 'rm -f "$tmp" "$checksum_tmp"' EXIT
curl -fsSL -o "$tmp" "$binary_url"
curl -fsSL -o "$checksum_tmp" "$checksum_url"

if command -v sha256sum >/dev/null 2>&1; then
  expected_name=$(basename "$tmp")
  (cd "$(dirname "$tmp")" && sha256sum -c "$checksum_tmp")
elif command -v shasum >/dev/null 2>&1; then
  expected=$(awk '{print $1}' "$checksum_tmp")
  actual=$(shasum -a 256 "$tmp" | awk '{print $1}')
  [ "$expected" = "$actual" ] || { echo "Checksum verification failed" >&2; exit 1; }
else
  echo "sha256sum or shasum is required" >&2
  exit 1
fi

chmod 0700 "$tmp"
mv "$tmp" "$aris_bin"
```

实际代码需要同时严格校验 checksum 文件格式，避免空值或多余内容被接受；校验成功后删除 checksum 临时文件并清理 trap。
